### `sim_controller.py`
Interactive flight simulator with real-time visualization and keyboard control.

**Controls:**

Left Stick (WASD):
- W/S - Throttle up/down
- A/D - Yaw left/right

Right Stick (Arrow Keys):
- ↑/↓ - Pitch forward/back
- ←/→ - Roll left/right

Other:
- R - Reset simulation
- Q - Quit
